Testing out Tanstack Start and Better Auth before I commit to using them on anything more significant

### Deps

```bash
nvm
pnpm
```

### Setup

```bash
nvm use
pnpm i
pnpm run init
```

### Dev

`pnpm dev`

Generated credentials are in [`.env`](.env)
